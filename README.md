# AI Scrap Metal Price Predictor

This project predicts future aluminium and copper prices using a simple machine learning model.

The model uses historical London Metal Exchange (LME) data to train a regression algorithm and forecast price trends.

## Features
- Aluminium LME price prediction
- Copper LME price prediction
- Linear regression machine learning model

## Technologies Used
Python  
Pandas  
Scikit-Learn  
NumPy
